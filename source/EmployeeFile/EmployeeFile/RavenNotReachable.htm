RavenDB instance not reachable

When initializing, RaccoonBlog could not reach the RavenDB server configured in web.config, please ensure that:

  • You have a server running in that address.
  • That it is reachable from the RaccoonBlog server.

Common causes for this error is to simply not have RavenDB running. RaccoonBlog requires RavenDB server to be running.

If you don't have RavenDB already, you can download it from http://ravendb.net/download, after which you can simply execute the Raven.Server.exe in the Server directory, to start it.
